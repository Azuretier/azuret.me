Hybrid Notes Desktop setup guide
Updated: 2026-04-13

1. Download the project bundle or clone the repository.
2. Open the project folder in a terminal.
3. Run: npm install
4. Run: npm run electron

What you get:
- Electron desktop app UI
- Local workspace storage
- Microsoft Edge tab import helper on Windows
- Analog capture, digital notes, bridge links, and JSON backup

Optional real-time tab capture:
1. Load the Edge extension from the hybrid-notes-edge extension bundle.
2. Keep Hybrid Notes Desktop running.
3. Turn on Auto send in the extension popup.
